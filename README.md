# Brightlane Verified Merchant Directory

A curated directory of verified affiliate merchant partners across software, 
education, pet care, health, printing, and lifestyle categories.

## Overview

This repository powers the Brightlane Verified Merchant Directory — a 
browsable, searchable collection of vetted merchant partners with affiliate 
links tracked through LinkConnector.

## Features

- 23+ verified merchant partners
- 7 product categories
- Search and filter functionality
- Multi-language support
- SEO optimized with structured data

## Structure

- `index.html` — Main directory page
- `blog/` — Supporting content
- `sitemap.xml` — Search engine sitemap
- `robots.txt` — Crawler directives
